'''------------------------------------------------------
【程序设计】
---------------------------------------------------------

题目：水仙花数

描述：水仙花数是指一个 3 位数，它的每个位上的数字的 3 
      次幂之和等于它本身（例如：1^3 + 5^3+ 3^3 = 153）,
      输出所有3位的水仙花数。

要求：

     输入格式：
     无

     输出格式：
     全部水仙花数，每行一个。


---------------------------------------------------------
注意：仅在注释标志之间填入所编写语句。
------------------------------------------------------'''
#**********Program**********
for i in range(100,1000):
    if (i//100)**3 + (i%10)**3 + (i//10%10)**3 == i:
        print(i)






#**********  End  **********
